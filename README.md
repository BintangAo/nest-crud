## Description

Basic CRUD using :
 - Nest
 - Neon + Drizzle

## Installation

```bash
$ npm install
```

## Creating .env file

```
DATABASE_URL=<YOUR_NEON_URL>
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
